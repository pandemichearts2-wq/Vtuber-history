- 詳細は `FA-SETUP.md` を参照してください。


## 画像保存抑止

公開FA画像には、右クリック・ドラッグ・スマートフォンの長押し保存を抑止する処理、透明保護レイヤー、サイト名と作者名の透かし、無断保存等を禁止する注意書きを追加しています。

ブラウザに表示される画像は、スクリーンショットや開発者ツールまで含めて完全に保存不能にはできません。この機能は一般的な保存操作をしにくくするための抑止策です。

## FA画像登録フォームの開閉

通常FA・成人向けFAとも、画像登録フォームは初期状態では非表示です。
各ページの「FA画像を登録する」ボタンを押すと表示されます。トップページの登録導線から開いた場合は、対象フォームが自動的に開きます。

- トップページの「FAを登録する」の横に「成人向けFAを登録する」を追加。
