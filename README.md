# Graduate History 軽量化版

ZIPを解凍し、中のWebサイト用ファイルをGitHubリポジトリ直下へアップロードしてください。

## 今回の変更

- FAはサーバーからランダム最大10件だけ取得
- FA一覧は400px画像、拡大時のみ1600px画像を読み込み
- 投稿画像はブラウザ内で長辺2000px以下・約2MB以下へ自動圧縮
- 動画登録ページのVTuber候補は、入力後に一致する最大20件だけ取得
- トップの動画・プロフィールは20件ずつ取得し「もっと見る」で追加
- Apps Scriptの検索・ページ取得結果を約5分キャッシュ

ドライブ容量の警告機能は入れていません。

## Apps Script

`AppsScript-Code-optimized.gs` へ置き換え、ウェブアプリを新しいバージョンとして再デプロイしてください。


## 管理画面のプロフィール修正

「詳細情報（JSON）」を廃止し、日本語ラベルとカテゴリ別の入力欄へ変更しました。登録済み項目のみを初期表示し、「空欄項目も表示」から未入力項目も追加できます。Apps Scriptの変更は不要です。
