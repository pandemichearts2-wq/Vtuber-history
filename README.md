# Graduate History 管理画面 件数バッジ色分け版

管理画面のタブ右上へ、SNSの通知バッジのような件数表示を追加しています。

- 申請管理：通常申請・動画申請・通常FA・成人向けFAの「確認待ち」合計を赤いバッジで表示
- お問い合わせ管理：
  - 赤い「未」バッジ＝未確認件数
  - 黄色い「中」バッジ＝対応中件数
- 各件数が0件なら、そのバッジだけ非表示
- 100件以上は `99+` と表示
- ログイン時、処理後、画面復帰時、60秒ごとに自動更新

## GitHub
`admin.html`、`admin.js`、`styles.css` を上書きしてください。

## Apps Script
`AppsScript-Code.gs` へ置き換えて保存し、既存ウェブアプリを「新バージョン」で再デプロイしてください。

`setupSheets` の再実行は不要です。
