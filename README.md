# Graduate History 管理画面 一括処理版

## 追加内容
- 通常申請（新規・追記・修正）
- 動画申請
- 通常FA
- 成人向けFA

上記すべての申請カードにチェックボックスを追加しました。

### 一括操作
- 一括チェック
- 一括チェック解除
- 選択した申請を一括承認
- 選択した申請を一括削除

一括削除は申請履歴を削除します。すでに公開済みのプロフィール・動画・FAそのものは削除しません。確認待ち・却下済みのFA申請を削除した場合、アップロード画像はGoogle Driveのゴミ箱へ移動します。

## 更新方法
1. GitHubのGraduate Historyリポジトリへ `admin.html` `admin.js` `styles.css` を上書きアップロードしてコミットします。
2. Graduate HistoryのApps Scriptコードを `AppsScript-Code.gs` に置き換えて保存します。
3. 「デプロイ」→「デプロイを管理」→鉛筆マーク→「新バージョン」→「デプロイ」を実行します。
4. `setupSheets` の再実行は不要です。
