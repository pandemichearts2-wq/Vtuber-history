# FA画像機能の追加手順

## 1. GitHub
このZIPを解凍し、中身をリポジトリ直下へ上書きアップロードします。

追加されるページ：
- `fanart/` 通常FA
- `adult-fanart/` 成人向けFA

## 2. Apps Script
`AppsScript-Code.gs`の内容で、現在のApps Scriptコードを置き換えます。

1. Apps Scriptで保存
2. `setupSheets`を一度実行し、権限を許可
3. 「デプロイ」→「デプロイを管理」→既存のウェブアプリを編集
4. バージョンを「新バージョン」にしてデプロイ

ウェブアプリURLが変わらなければ、`config.js`の変更は不要です。

## 3. 承認方法
スプレッドシートの `FA確認待ち` シートで対象行を選び、
Graduate Historyメニューから「選択中のFAを承認」を実行します。

- 通常FA → `公開FA`
- 成人向けFA → `公開成人向けFA`

承認した画像だけ、各ページのランダム表示に出ます。

## 注意
Google Workspaceの管理設定で「リンクを知っている全員」への共有が禁止されている場合、画像公開時にエラーになります。


## 画像保存抑止

公開FA画像には、右クリック・ドラッグ・スマートフォンの長押し保存を抑止する処理、透明保護レイヤー、サイト名と作者名の透かし、無断保存等を禁止する注意書きを追加しています。

ブラウザに表示される画像は、スクリーンショットや開発者ツールまで含めて完全に保存不能にはできません。この機能は一般的な保存操作をしにくくするための抑止策です。
